Fresh Takes Dashboard - Simple PRD Flow

Date: April 6, 2026

Simple Flow (What Is There vs What Should Be)

  1. Overview
    1. Beats

      What is there: Weekly beats snapshot and overview metrics.

      What should be there: "How many beats are we working on this week" and "How many beats were delivered last week (approved only)".

    2. Writer & POD output

      What is there: Writer/POD info is split across tabs.

      What should be there: In one place, show fresh takes delivered by each writer and each POD for last week (approved beats only).

    3. Production throughput

      What is there: Production data exists in Production tab.

      What should be there: Throughput per ACD directly visible in Overview.

    4. Full Gen AI

      What is there: Gen AI readiness is visible in analytics context.

      What should be there: Count of assets moved to Full Gen AI in Overview for the selected week.

  2. Editorial Funnel
    1. POD throughput view

      What is there: Last week / This week / Overview / Next week flows exist.

      What should be there: POD-level throughput ranking as default view.

    2. Writer drilldown

      What is there: Writer-level details are partially visible in different sections.

      What should be there: Writer-level throughput drilldown under each POD.

    3. Best POD indicator

      What is there: No single "best POD" callout.

      What should be there: A clear best POD badge based on throughput score for selected week.

  3. POD Wise
    1. Week filtering

      What is there: Performance and Tasks with week filter.

      What should be there: Keep current behavior, ensure filtering stays reliable for both views.

  4. Other Existing Tabs (No major new asks)
    1. Planner / Analytics / Production / Details

      What is there: Working and already in use.

      What should be there: Keep as is, only use them as source inputs for Overview and Editorial throughput summaries.

Very Short Build Plan

Step Action Output
